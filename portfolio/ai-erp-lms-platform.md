# AI-powered ERP/LMS Platform

## Overview

Led a unified Education ERP and LMS platform with an AI-powered chatbot and hybrid grading system. The platform brought attendance, fees, assignments, grading, communication, and academic workflows into one centralized product.

## Problem Statement

Educational institutions were using fragmented tools for attendance, fees, communication, assignments, grading, and student engagement. This created operational inefficiency, manual workload, low visibility, and inconsistent user experience.

## My Role

As Product Manager, I worked on product strategy, feature planning, PRDs, user stories, sprint execution, stakeholder alignment, and KPI tracking for the ERP/LMS platform.

## Key Responsibilities

- Defined product requirements for ERP and LMS workflows
- Created user stories for student, teacher, admin, and institution roles
- Created AI chatbot use cases for academic and operational queries
- Designed hybrid grading workflows using AI, peer, and teacher inputs
- Managed Jira sprint backlogs and Agile ceremonies
- Collaborated with engineering, UX, and institutional stakeholders
- Tracked user engagement, adoption, churn, NPS, and feature usage
